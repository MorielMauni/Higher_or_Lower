Overview
Welcome to the Higher Lower Game! This is a fun and engaging game where you guess whether the next Instagram account has more or fewer followers than the current one. This version uses Instagram follower counts as of September 2024.

How to Play
The game will display an Instagram account and its follower count.
You will be asked to guess if the next account has more or fewer followers than the current one.
Your goal is to guess correctly as many times as possible to achieve the highest score.
Features
Dynamic Content: The game uses real Instagram follower counts from September 2024.
Interactive Gameplay: Simple commands to input your guess and see results.
Score Tracking: Keep track of your score throughout the game.
